# Desarrollo de una Plataforma Web para la Difusión de Capacitaciones sobre Prácticas Agrícolas Sostenibles en el Cultivo de Caña Panelera, Usando Tecnologías Web

## Introducción

Colombia es uno de los principales productores de panela a nivel mundial, siendo el segundo país con mayor producción, alcanzando 1,2 millones de toneladas anuales. Sin embargo, a pesar de su importancia económica y social, el sector enfrenta múltiples desafíos, incluyendo la baja adopción de prácticas agrícolas sostenibles, el uso de técnicas tradicionales y la falta de plataformas educativas accesibles para los agricultores.

En respuesta a estas problemáticas, se propone el desarrollo de una plataforma web para la difusión de capacitaciones sobre prácticas agrícolas sostenibles en el cultivo de caña panelera. Esta plataforma permitirá a los agricultores acceder a información clave sobre métodos de producción eficientes y sostenibles, contribuyendo a mejorar la productividad y la sostenibilidad del sector.

## Objetivo

Desarrollar una plataforma web utilizando Angular con TypeScript, Firebase y Tailwind CSS para la difusión de capacitaciones sobre prácticas agrícolas sostenibles en el cultivo de caña panelera. La plataforma permitirá a los agricultores acceder a información actualizada, material educativo y recursos interactivos, con el objetivo de mejorar sus técnicas de producción y promover la sostenibilidad en el sector panelero.

## Alcance

- Implementación de una plataforma web interactiva que permita a los agricultores acceder a capacitaciones sobre prácticas agrícolas sostenibles.
- Uso de tecnologías web modernas como Angular, Tailwind y Firebase para la construcción de la plataforma.
- Integración de un sistema de gestión de usuarios para personalizar la experiencia de aprendizaje.
- Publicación de contenido educativo en diferentes formatos (videos, artículos, infografías).
- Facilitar el acceso a información actualizada sobre sostenibilidad agrícola y manejo eficiente de cultivos.

## Problemática

### Capacitaciones limitadas

Las capacitaciones en prácticas agrícolas sostenibles suelen ser presenciales, lo que impide que agricultores en áreas remotas accedan fácilmente a la información.

### Escasez de plataformas educativas

No existen suficientes plataformas dedicadas a la capacitación en prácticas agrícolas sostenibles, limitando la difusión del conocimiento y afectando la productividad del sector.

### Uso de técnicas tradicionales

Muchos agricultores continúan utilizando métodos tradicionales que no optimizan la productividad ni la sostenibilidad del cultivo, generando ineficiencia y degradación ambiental.

### Pérdidas por plagas

El manejo inadecuado de plagas genera pérdidas significativas en la producción agrícola, lo que afecta la seguridad alimentaria y la estabilidad económica de los productores.

## Impacto Esperado

- Mejora en la adopción de prácticas agrícolas sostenibles entre los agricultores de caña panelera.
- Aumento en la eficiencia y productividad del cultivo de caña panelera.
- Reducción del impacto ambiental asociado a la producción panelera.
- Creación de una red de aprendizaje accesible y continua para los agricultores.

## Tecnologías Utilizadas

- *Angular*: Framework principal para el desarrollo de la interfaz de usuario.
- *Tailwind CSS*: Framework de estilos para diseñar una interfaz moderna y responsiva.
- *Firebase*: Backend para la autenticación de usuarios y almacenamiento de datos.
- *HTML, CSS y JavaScript*: Tecnologías base para la estructura y funcionalidades del sitio web.
